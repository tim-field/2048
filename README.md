# 2048 Game

A React implementation of the popular 2048 game.

## Features

- Classic 2048 gameplay with arrow key controls
- Score tracking with high score persistence
- Animated tile movements
- Confetti celebration when reaching 2048
- Game timer

## Development

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build

# Type checking
npm run typecheck

# Format code
npm run format
```
